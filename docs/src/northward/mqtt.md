## 当前状态

::: warning
当前版本的 NG Gateway **尚未提供北向 MQTT 插件实现**（northward/mqtt）。  
该页面用于避免侧边栏入口指向 404，并提前说明 Roadmap。
:::

## 你现在可以用什么替代

- 如果你需要消息总线：优先使用 [`Kafka`](/northward/kafka/) 或 [`Pulsar`](/northward/pulsar/)
- 如果你需要 IoT 平台对接：使用 [`ThingsBoard`](/northward/thingsboard/)
